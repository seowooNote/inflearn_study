인터페이스(Interface)
    ; 객체의 타입을 정의할 때 사용할 수 있는 문법
        ; 함수도 객체 타입이라 정의 가능함

인터페이스와 함수 시크니처
    ; 변수에 함수에 할당하는 함수 표현식 형태에서만 인터페이스 타입을 사용할 수 있어 유의
        ; 인터페이스로는 함수 선언문을 타입을 지정할 수 있는 방법이 없음

인터페이스 병합
    ; 타입스크립트는 같은 이름의 인터페이스는 자동으로 합침
        ; 인터페이스 병합은 위치와 상관없이 병합이 이루어짐
            ; 타입스크립트 컴파일러는 인터페이스 병합을 먼저 수행한 다음, 타입 검사를 수행

인터페이스 상속
    ; 부모가 자식에게 물려주는 것
    ; 하위 집합이 상위 집합으로 부터 물려받는 것

인터페이스 다중상속
    ; 인터페이스를 조합해서 새로운 인터페이스를 정의할 때 사용

구조적 타이핑(structural typing)
    ; 객체의 형태에 기반하여 타입을 검사하는 시스템
        ; 타입스크립트는 객체의 타입 검사 객체 형태 기반하여 검사
            ; 객체의 형태만 일치하면 지정된 타입의 출처가 달라도 타입체크가 통과됨
