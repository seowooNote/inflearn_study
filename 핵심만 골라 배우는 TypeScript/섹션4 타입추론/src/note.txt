타입추론(type annotations)
    ; 값의 타입을 자동으로 추측하여 적용해주는 기능
        ; 변수에 타입을 지정하지 않아도 에러를 발생하지 않음
            ; 타입스크립트 컴파일러가 자동으로 변수에 할당된 값을 보고 타입을 추론해 줌(변수 선언과 동시에 값 할당)

null
    ; 개발자가 의도적으로 값을 비워둔 것

undefined
    ; 지금 당장 값이 할당되지 않음

리터럴 타입(Literal Type)
    ; 특정 값 자체를 타입으로 사용
        ; 기본 자료형
            ; const 
                ; 재할당을 할 수 없게 함(Symbol 제외) -> 리터럴 타입
        ; 참조 자료형
            ; 리터럴 타입으로 추론되지 않음
                ; 재할당이 불가능한 것이지 할당된 것에서는 값이 바뀔 수 있음
