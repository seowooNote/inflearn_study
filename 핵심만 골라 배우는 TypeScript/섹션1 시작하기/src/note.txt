컴파일(compile)
    ; 타입스크립트 파일을 자바스크립트 파일로 변환하는 과정
컴파일러(compiler)
    ; 컴파일에 사용하는 도구, 여기서는 typescript 패키지가 컴파일러

npm install typescript --save-dev
    ; TypeScript 컴파일러(tsc)를 프로젝트에 설치
        - TypeScript는 브라우저나 Node가 직접 실행 못함
        - 그래서 .ts -> .js 로 변환하는 컴파일러가 필요함
        => 그게 바로 TypeScript 컴파일러 tsc
        - --save-dev 의미
            - dev dependency (개발성 의존성) 즉, 개발할 때만 필요한 패키지

npx tsc src/index.ts
    ; TypeScript 파일을 JavaScript로 변환
        - src/index.ts -> src/index.js 생성
        - npx 의미 
            - 로컬 node_modules에 있는 프로그램 실행
        - 이 명령은 보통 잘 안 씀
                - 왜냐하면 프로젝트에서는 보통 tsconfig.json 을 사용해서 전체 컴파일하기 떄문

npx tsc --init
    ; TypeScript 설정 파일 생성
        - tsconfig.json 이 파일이 TypeScript 프로젝트의 핵심 설정 파일

npx tsc
    ; 프로젝트 전체 TypeScript 컴파일
        - tsconfig.json 기준으로 모든 .ts 파일을 .js로 변환

npm install ts-node --save-dev
    ; TypeScript를 컴파일 없이 바로 실행
        - node index.js O
        - node index.ts X
            => 그래서 ts-node 가 필요
