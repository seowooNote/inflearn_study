unknown 타입
    ; any 타입처럼 기본 자료형 및 참조 자료형의 모든 타입 허용

never 타입
    ; 절대로 발생하지 않는(을) 값 or 절대로 불가능한 경우의 값
        ; 함수의 실행이 끝나지 않거나(무한 루프) 값을 반환하지 않고 종료되는(예외 발생) 도달 불가능한 상태를 나타낼 때 주로 사용

유니온 타입(Union Type)
    ; 하나의 변수가 여러 개의 타입을 가질 수 있음
        ; |(파이프 기호 사용)

인터섹션 타입(Intersection Type)
    ; 자바스크립트의 AND 연산자와 비슷함
        ; &(엠펏센트 기호 사용)
